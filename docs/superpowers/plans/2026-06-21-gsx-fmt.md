## Goal

`gsx fmt` ‚Äî a canonical, zero-config gsx formatter: `internal/printer` (AST‚Üísource,
re-indents freely while preserving significant whitespace + gofmt's Go fragments)
plus the `gen fmt` command (`-l`/`-w`/`--check`). Validated by the render-faithfulness
and idempotence property tests over the corpus.

## The layout contract (forced by faithfulness)

`fmt` may only add/remove COSMETIC whitespace (what `wsnorm.Normalize` drops); it must
never alter significant whitespace. So `fmt = parse ‚Üí wsnorm.Normalize ‚Üí printer.Fprint`,
and the printer operates on the NORMALIZED AST. The layout rule:

- **Children list BLOCK iff** it has ‚â•1 **block-level** child AND **no surviving `Text`**
  node; **else INLINE** (single line, verbatim content).
  - Block-level = `*Element` (incl. component tags), `*Fragment`, `*IfMarkup`,
    `*ForMarkup`, `*SwitchMarkup`, `*GoBlock`, `*Doctype`, `*HTMLComment`.
  - Inline-level = `*Text`, `*Interp`.
  - BLOCK ‚Üí each child on its own line, indented one tab deeper; parent's `</tag>` (or
    `}`) on its own line. INLINE ‚Üí all children concatenated verbatim on one line
    between the parent's open/close.
  - Rationale: a surviving `Text` carries significant content/space; breaking around it
    would change rendering ‚Üí forbidden, so any list with `Text` stays inline-verbatim.
    `<title>{title}</title>` (only `Interp`) ‚Üí inline; `<div><p/><span/></div>` (only
    elements) ‚Üí block; `<p>a <b>x</b> b</p>` (`Text`+element) ‚Üí inline-verbatim.
- **Indentation:** tabs. **Attributes:** one line, source order, NO wrapping (design).
  **Top-level decls:** separated by ONE blank line. **Go fragments** (interp/attr
  exprs, params, recv, control-flow clauses, pipeline args, `GoBlock` code, `GoChunk`)
  ‚Üí `go/format` (expressions wrapped as needed to format, then unwrapped).
- **Preserve contexts** (`pre`/`textarea`/`script`/`style`): emit the element's
  children **verbatim** (the `Text` is already preserved by `Normalize`) ‚Äî NO added
  indentation inside; treat as inline-verbatim regardless of the rule above.

## Task 1: `internal/printer` (AST ‚Üí canonical gsx source)

`internal/printer/printer.go`: `func Fprint(w io.Writer, f *ast.File) error`. Assumes a
normalized AST (documented; `gen fmt` Normalizes first). Emits:
- `package <P>\n\n` then each `Decl` (GoChunk / Component) separated by a blank line.
- **GoChunk**: `go/format.Source(Src)` (it's top-level Go decls), emitted verbatim.
- **Component**: `component [<Recv> ]<Name>(<Params>) {` newline, body (the layout rule,
  indent 1), `}`. `Recv`/`Params` formatted via go/format (synthesize `func <Recv>
  _m(<Params>){}`, extract). A nullary `()` stays `()`.
- **Markup** (a small `printMarkup`/`printChildren(list, indent)` recursion):
  - `*Element`: `<tag` + attrs (printAttr each, space-separated) + (`/>` if Void & no
    children else `>`); then children per the BLOCK/INLINE rule; then `</tag>` (block:
    own line; inline: directly). Preserve-tag elements ‚Üí children verbatim.
  - `*Interp`: `{ <gofmt-expr>[?] }` for a bare expr; `{ <seed> |> <stage> |> ‚Ä¶ }` when
    `Stages` non-empty (each stage `name` or `name(args)`, trailing `?` per stage).
  - `*Text`: the value verbatim (it's normalized).
  - `*Fragment`: `<>` + children (block/inline) + `</>`.
  - `*IfMarkup`: `{ if <cond> {` children `}` [` else if ‚Ä¶`/` else {` ‚Ä¶ `}`] ` }`.
    `*ForMarkup`: `{ for <clause> {` body `} }`. `*SwitchMarkup`: `{ switch [<tag>] {`
    cases (`case <list>:` / `default:` + body) `} }`. `*GoBlock`: `{{ <gofmt-stmts> }}`.
  - `*Doctype` / `*HTMLComment`: verbatim text.
- **printAttr** per kind: `StaticAttr` `name="value"`; `BoolAttr` `name`; `ExprAttr`
  `name={<gofmt-expr>[?]}` (+ pipeline stages like Interp); `SpreadAttr` `{...<expr>}`;
  `ClassAttr` `class={ <part>[: <cond>], ‚Ä¶ }`; `CondAttr` `{ if <cond> { <attrs> } [else
  ‚Ä¶] }`; `MarkupAttr` `name={ <markup> }` (the Value markup printed inline or block).
- Go-fragment formatting helpers: `fmtExpr(src)` (wrap `var _ = (src)`, go/format,
  extract), `fmtStmts(src)` (wrap in a func body), `fmtParams`/`fmtRecv`. On a format
  ERROR (malformed Go), fall back to the trimmed verbatim source (fmt must never fail
  on parseable gsx ‚Äî a Go fragment that doesn't gofmt is left as-is).
- Unit tests (`printer_test.go`): hand-written `input ‚Üí expected canonical output` for
  each construct (element block/inline, interp, pipeline, all attr kinds, control flow,
  fragment, components, GoChunk, pre/textarea verbatim, nested). Plus a per-input
  idempotence check (`Fprint(Normalize(parse(Fprint(...)))) == Fprint(...)`).

## Task 2: `gen fmt` command + property tests over the corpus

- `gen/fmt.go`: `func runFmt(stdout, stderr io.Writer, args []string) int` ‚Äî flags:
  default = rewrite files in place (`-w` implied? follow gofmt: default writes to stdout
  for a single file, but for gsx keep it simple ‚Äî **default lists nothing and rewrites
  in place is dangerous; use gofmt semantics**: no flag ‚Üí print formatted source of each
  file to stdout; `-w` ‚Üí write in place; `-l` ‚Üí list files that differ; `-d` ‚Üí diff.
  Pick the gofmt-faithful set: `-l`, `-w`, `-d`, default‚Üístdout). Operate on the path
  args (files or dirs, recurse `.gsx` reusing `discoverDirs`-style walk, or accept file
  args). For each `.gsx`: read ‚Üí `parser.ParseFile` ‚Üí `wsnorm.Normalize` ‚Üí
  `printer.Fprint` ‚Üí compare/emit/write. A parse error ‚Üí stderr, exit 1 (per file),
  continue others; exit 1 if any file errored or (`-l`/`-d`) any differs.
- Wire `case "fmt":` into `runConfig` (gen/main.go); add `fmt` to `printUsage`.
- **Property tests** (`internal/printer` or a new `gen`/printer test, over the corpus
  case `input.gsx` files ‚Äî load them via the corpus loader or a glob of
  `internal/corpus/testdata/cases/**/*.txtar` extracting the `input.gsx` section):
  - **Faithfulness:** for each input `S`, `Normalize(parse(fmt(S)))` deep-equals
    `Normalize(parse(S))` (the rendered AST is identical) ‚Äî OR, stronger, generate from
    both and structurally compare. The AST-equality form is cheaper and sufficient;
    implement an AST structural-equal (ignore spans) or reuse `ast.Fprint` dumps:
    `astDump(Normalize(parse(fmt(S)))) == astDump(Normalize(parse(S)))`.
  - **Idempotence:** `fmt(fmt(S)) == fmt(S)` (byte-identical) for every input.
  - **Re-parse safety:** `fmt(S)` always re-parses without error.
  Run over ALL 148 corpus inputs ‚Äî this is the real validation that the printer is
  faithful + idempotent across every language construct in the corpus.
- Tests: `runFmt` exit codes (`-l` on an unformatted file ‚Üí 1 + lists it; `-w` rewrites;
  default ‚Üí stdout; a parse-error file ‚Üí 1). Use temp files.

## Risks
- **Faithfulness on mixed content** ‚Äî the BLOCK/INLINE rule is the crux; the corpus-wide
  property test is the guard. If any corpus input fails faithfulness, the rule needs
  refining (likely a list wrongly classified block/inline).
- **Go-fragment formatting** ‚Äî expressions aren't whole files; wrap/extract carefully and
  fall back to verbatim on error (never fail fmt on parseable gsx).
- **Idempotence** ‚Äî `astDump` equality + byte-identical second pass; a non-idempotent
  construct (e.g. inconsistent spacing) surfaces immediately.
